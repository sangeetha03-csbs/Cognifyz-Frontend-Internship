# Cognifyz Front-End Development Internship Project

## 📌 Project Overview
This project was developed as part of the **Cognifyz Technologies Front-End Development Internship**. It demonstrates the use of HTML, CSS, JavaScript, Bootstrap, API integration, responsive design, form validation, and SCSS.

---

## 🚀 Features

- ✅ Basic HTML webpage
- ✅ Responsive design using CSS Media Queries
- ✅ Interactive button to change background color
- ✅ API Integration using JSONPlaceholder
- ✅ Contact form with client-side validation
- ✅ Bootstrap components (Cards, Alerts, Buttons)
- ✅ SCSS variables and nested styles
- ✅ Mobile-friendly design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- SCSS (Sass)
- JSONPlaceholder API
